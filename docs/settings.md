## Design Philosophy

### Core Principles

1. **Abstract the engine** - Users shouldn't know or care about Aria2 internals
2. **Task-oriented organization** - Group settings by what users want to DO, not by technical category
3. **Progressive disclosure** - Simple defaults shown first, power user options hidden but accessible
4. **Provider-first cloud setup** - Visual wizard-based configuration with provider icons
5. **Premium-first downloads** - Debrid services are first-class citizens, not an afterthought

### User Personas

| Persona | Needs | Settings Focus |
|---------|-------|----------------|
| **Casual User** | Just download files quickly | Overview + Downloads basics |
| **Cloud Power User** | Auto-upload to multiple clouds | Cloud Storage wizard |
| **Debrid User** | Use AllDebrid/Real-Debrid for everything | Premium Services |
| **Torrent User** | Fine-tune seeding and privacy | Torrents section |
| **Network Admin** | Proxy, connection limits, security | Network section |
